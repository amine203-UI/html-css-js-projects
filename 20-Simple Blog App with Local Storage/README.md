 Simple Blog App

This is a simple blog application that allows users to create and delete blog posts.

 Features

- Create Post: Add new blog posts with a title and content.
- Delete Post: Remove existing blog posts.

 Usage

You can use the app to:
- Write and publish new posts.
- Delete posts you no longer want.

 Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage (for saving posts in the browser)

Getting Started

1. Clone the repository.
2. Open the `index.html` file in your browser.
