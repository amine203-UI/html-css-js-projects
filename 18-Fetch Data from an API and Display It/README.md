
 Weather App


 Overview

A simple app that fetches real-time weather data from a public API and displays it in a clean, user-friendly format.


- Features

Search weather by city

Shows temperature, humidity, wind speed, and description

Easy-to-use interface



- Tech Stack

HTML, CSS, JavaScript

API: OpenWeatherMap



- Setup

Get an API key from OpenWeatherMap

Replace YOUR_API_KEY in script.js

Open index.html in your browser
