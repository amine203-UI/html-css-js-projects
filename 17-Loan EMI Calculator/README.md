
 Loan EMI Calculator 

A simple and modern Loan EMI Calculator built with HTML, CSS, and JavaScript.

---

 Features
- Clean, responsive design   
- Calculates EMI in US Dollars ($)  
- Lightweight and easy to use   



 Formula
 
\[EMI = \frac{P \times R \times (1+R)^N}{(1+R)^N - 1}\]

- P = Principal  
- R = Monthly Interest Rate (Annual รท 12 รท 100)  
- N = Loan Tenure in Months  

---

 Usage
1. Open `index.html` in your browser.  
2. Enter loan details and click Calculate EMI.  
3. The result will display instantly.
