 Number Guessing Game

This is a simple and interactive Number Guessing Game built with HTML, CSS, and JavaScript.


 How to Play
- The game randomly selects a number between 1 and 100.
- Enter your guess in the input field and click the "Submit" button.
- The game will tell you if your guess is too high, too low, or correct.
- The number of attempts is displayed under the feedback message.
- When you guess the correct number, a congratulatory message appears and you can play again by clicking the "Play Again" button.

 Features
- Modern UI: Glassmorphism background, orange accent colors, and smooth transitions.
- Responsive Design: The game is centered and adapts to different screen sizes.
- Local Storage: The last input value and feedback message are saved, so refreshing the page will not lose your progress.
- Footer: A fixed footer with a white line and "All rights reserved" message.

 Files
- `index.html` — Main HTML structure for the game.
- `style.css` — Styles for layout, colors, effects, and responsiveness.
- `script.js` — Game logic and local storage handling.
